# ML Lab

Experimental notebooks and dependencies for recommendation/search prototyping.

## Contents

- `notebooks/collaborativefiltering.ipynb`
- `notebooks/datasetreduce.ipynb`
- `notebooks/recommendation.ipynb`
- `notebooks/search.ipynb`
- `requirements.txt`

## Purpose

This folder is intended for offline experimentation and model/prototype exploration, separate from production service runtime paths.

## Setup

```bash
cd ml-lab
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Then open notebooks with Jupyter.
